Acesse a **Corretora de Imóveis** diretamente no navegador:

## 💻 Sobre o Projeto

Uma aplicação web moderna para gerenciar e exibir propriedades imobiliárias. Desenvolvida com React, TypeScript e Vite, oferecendo uma experiência rápida e responsiva.

- **Página Inicial Dinâmica**: Hero section, sobre a empresa e propriedades em destaque
- **Catálogo de Propriedades**: Visualização de imóveis em destaque com cards informativos
- **Página de Detalhes**: Informações detalhadas de cada propriedade
- **Contato**: Formulário de contato para clientes interessados
- **WhatsApp Integrado**: Botão flutuante para contato direto via WhatsApp
- **Design Responsivo**: Interface adaptada para desktop, tablet e mobile
- **Roteamento**: Navegação entre páginas usando React Router

## 🛠️ Tecnologias

- **React 19.2.7** - Biblioteca UI
- **TypeScript 6.0** - Tipagem estática
- **Vite 8.1.0** - Build tool rápido
- **React Router DOM 7.18.1** - Roteamento
- **ESLint** - Linting de código

## 📋 Pré-requisitos

- Node.js (versão 18 ou superior)
- npm ou yarn

## 🚀 Instalação e Uso

### Instalação das dependências

```bash
npm install
```

### Desenvolvimento

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

O aplicativo estará disponível em `http://localhost:5173` (porta padrão do Vite).

### Build para Produção

```bash
npm run build
```

### Build para Produção (Vercel)

```bash
npm run build:vercel
```

### Preview da Build

```bash
npm run preview
```

### Lint

Verifique a qualidade do código:

```bash
npm run lint
```

## 📁 Estrutura do Projeto

```
src/
├── components/          # Componentes reutilizáveis
│   ├── About/          # Seção sobre a empresa
│   ├── Contact/        # Formulário de contato
│   ├── FeaturedProperties/ # Propriedades em destaque
│   ├── Footer/         # Rodapé
│   ├── Header/         # Cabeçalho/Navegação
│   ├── Hero/           # Seção hero
│   ├── PropertyCard/   # Card de propriedade
│   └── WhatsAppButton/ # Botão flutuante WhatsApp
├── pages/              # Páginas da aplicação
│   └── PropertyDetailsPage/ # Detalhes de propriedade
├── data/               # Dados estáticos
│   └── properties.ts   # Lista de propriedades
├── styles/             # Estilos globais
├── App.tsx             # Componente principal
└── main.tsx            # Entry point
```

## 🔀 Rotas

- `/` - Página inicial (home)
- `/imovel/:id` - Detalhes da propriedade (filtrado por ID)

## 🎨 Customização

### Adicionar Propriedades

Edite o arquivo `src/data/properties.ts` para adicionar, remover ou modificar propriedades.

### Estilização

Cada componente possui seu próprio arquivo CSS. Os estilos globais estão em `src/styles/`.

## 📱 Recursos Principais

### Header
Navegação principal com links para as seções da página.

### Hero Section
Banner inicial com destaque da empresa.

### Sobre
Informações sobre a corretora de imóveis.

### Propriedades em Destaque
Exibe uma seleção de propriedades principais com cards interativos.

### Contato
Formulário para que interessados possam entrar em contato.

### WhatsApp Button
Botão flutuante fixo que permite contato direto via WhatsApp.

### Footer
Informações complementares e links adicionais.
